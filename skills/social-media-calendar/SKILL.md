# Social Media Calendar

Produce a ready-to-schedule calendar: dated posts, per-platform copy, and a repeatable content mix — not a vague list of "post ideas."

## Workflow

1. **Establish the inputs.** From context, determine: platforms, posting frequency, time horizon (default: 2 weeks), business goal (awareness, leads, community), and any events/launches to build around.
2. **Set the content mix.** Use a default ratio of 40% value/education, 25% engagement (questions, polls), 20% social proof/behind-the-scenes, 15% promotion. Adjust if the user states a goal — never exceed 30% direct promotion or engagement decays.
3. **Write real posts, not placeholders.** Every calendar slot gets complete, publishable copy. Respect platform norms:
   - **LinkedIn**: hook line first (it's all readers see before "…more"), short paragraphs, no more than 3 hashtags, 150–300 words.
   - **X/Twitter**: under 280 characters, or a numbered thread with a strong first tweet.
   - **Instagram**: caption with a hook, line breaks, CTA to comment/save, 3–5 hashtags, plus an image/reel description for the designer.
   - **TikTok/Reels**: a 15–30 second script with spoken hook in the first 2 seconds and on-screen text notes.
   - **Facebook**: conversational, 40–120 words, question or link CTA.
4. **Assemble the calendar.** Output a table: date, platform, content pillar, full copy (or link to it below), asset needed, CTA.
5. **Add the operating notes.** Best posting times per platform, what to measure weekly (saves/shares over likes), and which posts to boost if budget exists.

## Output format

A calendar table first (date, platform, pillar, hook/first line, asset), then the full copy for each post under a dated heading. Never ship a slot that says "TBD."

## Quality bar

- Hooks must create curiosity or state a bold claim — cut any opener that could belong to any company.
- Reuse one core idea across platforms by reshaping it, not copy-pasting it.
- Include one engagement-bait-free question post per week; genuine questions outperform "engagement hacks."
